# Bias-Resilient Ensemble Models for Fair Decision Systems

This project implements fairness-aware ensemble learning techniques to reduce bias in machine learning models across multiple real-world domains.

---

## Datasets Used

- COMPAS (Criminal Justice)
- Adult Income (Socio-economic)
- Law School (Education)

---

## Models and Techniques

- Logistic Regression
- Random Forest
- XGBoost

### Ensemble Methods
- Voting Ensemble
- Stacking Ensemble
- Fairness-Weighted Ensemble (Proposed)
- Fairness-Aware Stacking (Advanced)

---

## Evaluation Metrics

- Accuracy
- F1 Score
- Statistical Parity Difference (SPD)
- Equal Opportunity Difference (EOD)

---

## Key Insights

- Fairness improvements are dataset-dependent
- There is a clear trade-off between fairness and accuracy
- Ensemble methods can balance bias and performance
- Advanced fairness techniques reduce bias but may impact performance

---

## Conclusion

This project demonstrates that fairness-aware ensemble learning can effectively reduce bias in high-stakes decision systems, but must be carefully tuned depending on the dataset and application.
